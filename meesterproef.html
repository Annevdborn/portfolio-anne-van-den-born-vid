Project Details

Mijn meesterproef is een persoonlijk interactief webproject rondom het thema "Music Feels Different at Night". De website onderzoekt hoe muziek anders kan aanvoelen wanneer je er 's avonds of 's nachts naar luistert, gecombineerd met persoonlijke herinneringen en verhalen.

Klant Hogeschool van Amsterdam
Jaar 2026
Diensten Web Design (UI/UX)
Software VS Code & Figma

Proces

Het project begon met uitgebreid conceptonderzoek. Ik werkte mijn idee uit, verzamelde inspiratie en selecteerde muziek die voor mij een bijzondere betekenis heeft. Bij ieder nummer schreef ik persoonlijke verhalen die de beleving versterken.

Gedurende het project heb ik meerdere feedbackrondes doorlopen. Op basis van deze feedback heb ik zowel het concept als de vormgeving verder ontwikkeld. De website kreeg steeds meer een nachtelijke sfeer door het gebruik van kleur, animaties, typografie en interactie.

Het eindresultaat is een zeer persoonlijk digitaal verhaal waarin muziek, emotie en webdesign samenkomen. Juist doordat het project zo dicht bij mezelf ligt, voelt het als een ontwerp dat echt mijn eigen stijl en visie laat zien.
